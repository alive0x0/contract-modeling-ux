# Design System — Cigna Canvas Tokens

Token files for the Canvas design system. Used to generate accurate mockups and prototypes.

## Files

| File | Contents |
|---|---|
| `Semantic Tokens.zip` | Cigna Light Mode, Cigna Dark Mode, Evernorth Light Mode, Evernorth Dark Mode semantic tokens |
| `Option Tokens.zip` | Base color palette (Light, Dark, Spacing, Border Radius) |

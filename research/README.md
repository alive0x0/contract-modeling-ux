## Key Findings

- **Combined feature is a lifesaver** for national negotiations but only power users know how to use it
- **Excel tracker dependency** — analysts maintain external spreadsheets to track scenario IDs by region
- **Zero provenance** — a month after combining, analysts cannot identify which source IDs were used
- **Habituation risk** — Skander has normalized workarounds; pain is understated in interviews
- **Feature awareness gap** — many existing features (uncertainty analysis) are underused, not unknown

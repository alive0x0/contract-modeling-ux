Phase 1 — What's New in This Redesign

🔍 Global Search Bar NEW
Single search bar replaces 13 column-level filters. Searches across name, ID, contract office, and tags simultaneously. Eliminates the need to know which column to filter.
● Run Status — Column 1 ENHANCED
Status moved from column 10–11 to column 1 (right after checkbox). Color-coded dots — no more scanning to the far right to answer "did my scenario finish?"
ASC Facility Claim Type Tags NEW
System-generated tags for Facility, Professional, and ASC claim types. Addresses Skander's blind spot — no column currently distinguishes ASC from Facility scenarios.
TX-Neg-Group-4 User Custom Tags NEW
Freeform tags users can apply to any scenario. Replaces the naming conventions ([COPY], manual prefixes) and Excel workarounds for grouping by negotiation group or project phase.
RSH Sync Status NEW
Visible indicator showing whether a scenario has been pushed to RSH and if the RSH copy is still current. States: Synced ✓, Stale (source updated since last push), Unknown. Eliminates Excel tracking.
⛓ Combined Combined Badge NEW
Visual identifier in the name column distinguishes combined scenarios from individual ones. Prevents confusion when scanning the list — no longer requires opening each row to check its type.
⛓ Combine Selected — Bulk Action NEW
Multi-select rows with checkboxes, then hit "Combine Selected" to initiate a combined scenario from the search page. Eliminates leaving the page, navigating to the Combined Scenario page, and manually typing IDs.
Column Visibility Controls ENHANCED
Toggle which columns are visible. Status, Claim Type, RSH, and Tags are on by default. LOB and Last Run are off by default (most users don't need them immediately). Saves horizontal scroll.

# Contract Modeling — UX Research & Design

UX wireflows, research insights, design tokens, and prototypes for the Contract Modeling tool.

## Repository Structure

```
design-system/          ← Cigna Canvas token files
  Semantic Tokens.zip   ← Light/Dark mode semantic tokens (Cigna + Evernorth)
  Option Tokens.zip     ← Base color palette, spacing, border radius
  README.md             ← Key token reference

wireflows/
  combined-scenarios/
    combined_scenario_flowchart.html   ← Business flow (4 phases, 3 views)
    combined_scenario_wireflow.html    ← Interactive 4-step UX prototype
    screenshots/                       ← Step-by-step screen captures

research/
  README.md             ← Interview index and key findings
```
